## Overview

**cPGuard X** is a modern hosting control and security management platform designed to simplify server and website administration while maintaining strong security protections.  

The platform integrates advanced security features from the **cPGuard ecosystem** along with hosting management capabilities such as database management, file management, automated backups, and CMS deployment.

It is built for **developers, hosting providers, IT administrators, and businesses** that require centralized control of their web infrastructure with enterprise-grade protection.

---


## Scope

The scope of **cPGuard X** includes centralized server administration, website deployment, security management, and application lifecycle management across one or more servers.

### Key Functional Areas

#### Centralized Server Management
Manage multiple servers, websites, and configurations from a single dashboard.

#### Security Integration
Built-in protection using cPGuard security features such as:
- Malware scanning  
- Firewall protection  
- Threat mitigation  

#### Web Stack Management
Native support for web servers such as:
- **Nginx**
- **OpenLiteSpeed**

with optimized configurations.

#### CMS and Application Deployment
One-click deployment and automated updates for platforms such as:
- **WordPress**


#### File and Database Management
Integrated tools for:
- File editing
- File uploads
- Database creation
- phpMyAdmin access

#### Backup and Recovery
Automated backup scheduling with:
- Version retention
- One-click restoration
