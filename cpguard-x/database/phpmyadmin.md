**phpMyAdmin** allows you to manage your databases through a web-based interface.

From the **cPGuard X** dashboard, click on **Website**, then go to **Database**.

![cPGuard X websites](../../assets/img/websites/image-mig-19.png)

![cPGuard X websites](../../assets/img/websites/image-mig-20.png)

- Click **phpMyAdmin**.

![cPGuard X websites](../../assets/img/websites/image-img-22.png)

- Select the  **database**.

![cPGuard X websites](../../assets/img/websites/image-mig-23.png)

You can manage your databases efficiently through **phpMyAdmin**, where you can:

- View and manage all databases and tables  
- Run SQL queries  
- Import and export databases  
- Edit table data and structure  
- Manage indexes and relationships
